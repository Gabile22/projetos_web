Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem ordem rígida, por exemplo ingredientes em uma receita.

A tag li (list itens - itens da lista) representa um item que faz parte de uma lista.

Exemplo de lista

  • primeiro item
  • segundo item
  • terceiro item
    • subitem

Listas ordenadas

A tag ol (ordered lista - lista ordenada) representa uma lista de itens numerados sequencialmente.

Exemplo de lista ordenada

  1. Primeiro item
  2. Segundo item

Atributos

reversed (ordem reserva)
Exibe os itens da lista ordenada em ordem reserva

  • Item mais importante
  • Item menos importante
    •

    Start (início)
    específico o valor inicial para numeração da lista.

    1. Item 5
    2. Item 6

    type (tipo)
    indica o tipo de numeração:
    a - indica letras minúsculas
    A - indica letras maiúsculas
    i - indica algarismos romanos minúsculos
    I - indica algarismos romanos maiúsculos

    1. Primeiro item
    2. Segundo item
